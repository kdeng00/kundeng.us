## Required
* NodeJs = v16.7.0
* NPM & NPX
* ReactJS


## Getting started
1. Clone repo
```Bash 
git clone https://github.com/kdeng00/kundeng.us
```

2. Change directories to the cloned repository

3. Make sure React is installed
```Bash
npx install create-react-app client
npm install react-bootstrap bootstrap
```

4. Start app
```Bash
cd client
npm start
```
