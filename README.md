## Required
* Python >= 3.6.9
* Django >= 3.0.7
* Pip3 >= 20.1
* Virtualenv >= 15.1.0
* NodeJs = v16.7.0
* ReactJs


## Running
1. Clone repo
```Bash 
git clone https://github.com/kdeng00/kundeng.us
```

2. Change directories to the cloned repository

3. Make sure React is installed
```Bash
npx install create-react-app
```

4. Start app
```Bash
npm run
```
